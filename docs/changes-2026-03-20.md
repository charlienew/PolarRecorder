# Changes — 2026-03-20

## 1. Accelerometer settings dialog is now scrollable

**File:** `ui/dialogs/DeviceSettingsDialog.kt`

The `DataTypeSettingsDialog` content column was not scrollable, causing the lower options (e.g. ±4g, ±8g range values) to be clipped on small screens or when many options are present. Added `verticalScroll(rememberScrollState())` to the column so all settings are reachable.

## 2. PPG data preview shows first channel

**File:** `managers/RecordingUtils.kt`

The live data preview for PPG was reading the last channel (`channelSamples.lastOrNull()`). On the Polar Loop this is always the 3rd channel, which outputs a constant value of 3. Changed to `channelSamples.firstOrNull()` so the preview reflects channel 0, which carries the actual PPG signal.
